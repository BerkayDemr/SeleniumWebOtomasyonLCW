# SeleniumWebOtomasyon
 E-ticaret web sitesinde sepete ürün ekleme.

Proje Açıklaması

- Proje Maven projesi olarak, Java programlama dilinde yazılmıştır.
- Selenium ve JUnit kütüphanelerini kullanılmıştır.
- Proje OOP (Object Oriented Programming) prensiplerine uygun yazılmıştır.
- Page Object Pattern kullanılmıştır.
- Locator’lar sayfanın üst bölümünde tanımlanmıştır.
- Chrome browser kullanılmıştır.
- Test bittikten sonra driver kapatılmıştır.
- Sayfaların doğruluğunun kontrolü için her adımda assertion kullanılmıştır.

Senaryo Adımları

1. https://www.lcwaikiki.com/tr-TR/TR adresine gidilir.
2. Herhangi bir kategori sayfasına gidilir.
3. Herhangi bir ürün sayfasına gidilir.
4. Ürünü sepete eklenir.
5. Sepet sayfasına gidilir.
6. Anasayfaya geri dönülür.
